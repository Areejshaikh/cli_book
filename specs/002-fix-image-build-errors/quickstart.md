# Quickstart Guide: Fix Image Build Errors and Create Text+Emoji UI

## Development Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Key Files

- `src/pages/index.js` - Main homepage component with the new text+emoji UI
- `src/pages/index.module.css` - Styles for the homepage

## Running Tests

To build the site for production:
```bash
npm run build
```

To serve the built site locally:
```bash
npm run serve
```

## Deployment

The site is configured for deployment to GitHub Pages. After pushing changes to the main branch, the site will automatically build and deploy.
