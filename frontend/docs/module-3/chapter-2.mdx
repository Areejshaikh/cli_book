# Isaac ROS: hardware-accelerated VSLAM & navigation

## Learning Objectives
- Learn about Isaac ROS for hardware-accelerated VSLAM
- Understand navigation capabilities
- Implement basic VSLAM techniques

## Introduction

Isaac ROS is a collection of packages that bring NVIDIA's hardware acceleration to ROS 2. It provides optimized implementations for perception and navigation tasks, particularly Visual Simultaneous Localization and Mapping (VSLAM).

## Isaac ROS Capabilities

Isaac ROS packages are designed to run on NVIDIA hardware to provide accelerated performance for perception and navigation tasks.


## Summary
- Isaac ROS provides hardware acceleration
- Optimized for perception and navigation
- Uses NVIDIA hardware for acceleration
