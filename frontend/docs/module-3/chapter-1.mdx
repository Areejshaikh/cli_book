# Isaac Sim: photorealistic simulation & synthetic data

## Learning Objectives
- Understand Isaac Sim for photorealistic simulation
- Learn about synthetic data generation
- Implement basic simulation techniques

## Introduction

Isaac Sim is NVIDIA's next-generation robotics simulator based on Pixar's Universal Scene Description and NVIDIA's Omniverse tech. It provides photorealistic simulation capabilities and synthetic data generation for training AI models.

## Features of Isaac Sim

Isaac Sim offers several key features including photorealistic rendering, physics simulation, and synthetic data generation tools.


## Summary
- Isaac Sim provides photorealistic simulation
- Synthetic data generation accelerates AI training
- Based on USD and Omniverse technology
