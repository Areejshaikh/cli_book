# Simulate physics, gravity, and collisions in Gazebo

## Learning Objectives
- Understand how to simulate physics in Gazebo
- Learn about gravity and collision simulation
- Implement basic physics simulations

## Introduction

Gazebo is a powerful physics simulator that provides realistic simulation of robots in complex indoor and outdoor environments. It provides high-quality graphics and physics simulation and serves as a stand-in for the real world.

## Physics Simulation in Gazebo

Gazebo uses Open Dynamics Engine (ODE), Bullet Physics, and Simbody as its physics engines. These engines handle the complex calculations required for realistic physics simulation.


## Summary
- Gazebo provides realistic physics simulation
- Multiple physics engines available
- Essential for robotics development and testing
