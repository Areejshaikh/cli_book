# Capstone: Autonomous humanoid executes voice commands, navigates, identifies, and manipulates objects

## Learning Objectives
- Integrate all learned concepts into a capstone project
- Implement autonomous humanoid behaviors
- Combine voice recognition, navigation, and manipulation

## Introduction

This capstone project combines all the concepts learned in the previous modules. The autonomous humanoid will execute voice commands, navigate through environments, identify objects, and manipulate them using the techniques learned throughout the course.

## Capstone Project Components

The project integrates:
- Voice recognition using Whisper
- Natural language processing using LLMs
- Navigation using Nav2
- Object identification and manipulation


## Summary
- Capstone integrates all learned concepts
- Combines multiple technologies
- Demonstrates complete humanoid autonomy
