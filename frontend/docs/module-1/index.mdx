# The Robotic Nervous System (ROS 2)

## Module Overview

This module introduces the fundamentals of ROS 2 (Robot Operating System 2), which provides a flexible framework for writing robot software. You'll learn about nodes, topics, services, and how to control ROS using Python agents.

## Learning Outcomes

By the end of this module, you will:
- Understand the ROS 2 architecture and its core components
- Be able to create and run basic ROS 2 nodes
- Know how to implement communication between nodes using topics and services
- Understand the Unified Robot Description Format (URDF) for humanoid robots
